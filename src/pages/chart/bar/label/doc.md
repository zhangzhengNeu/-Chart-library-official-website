### bar-label

折线图数据点标签

**属性配置**

| 属性名       | 说明                                                                                                | 类型      | 默认值                            |
| ------------ | --------------------------------------------------------------------------------------------------- | --------- | --------------------------------- |
| showLabel    | 是否显示标签                                                                                        | `boolean` | true                              |
| showPercentage | 是否显示百分比                                                                                    | `boolean` | false                              |
| position     | 标签位置参考：[label.position](https://echarts.apache.org/zh/option.html#series-bar.label.position) | `string`  | 柱状图`inside` 条形图`insideLeft` |
