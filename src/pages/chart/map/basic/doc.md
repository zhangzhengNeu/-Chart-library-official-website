## 地图

---

### map-basic

基础地图

**属性配置**

| 参数        | 说明           | 类型     | 默认值 | 是否必填 |
| ----------- | -------------- | -------- | ------ | -------- |
| mapType     | 地图 adcode    | `number` | 100000 | 是       |
| areaColor   | 地图区域的颜色 | `string` | #eee   | 否       |
| borderColor | 描边颜色       | `string` | #fff   | 否       |
