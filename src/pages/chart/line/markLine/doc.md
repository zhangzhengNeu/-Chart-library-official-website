### line-markLine

显示标记线，可以是最大值、最小值、平均值

**属性配置**

| 属性名 | 说明       | 类型                        | 默认值    |
| ------ | ---------- | --------------------------- | --------- |
| type   | 标记线位置 | 'min' \| 'max' \| 'average' | 'average' |
| name   | 标记线名称 | `string`                    | '平均值'  |
