### pie-ringIndicator

环形指标卡

**属性配置**

| 属性名 | 说明     | 类型                 | 默认值   |
| ------ | -------- | -------------------- | -------- |
| name   | 中心文字 | `string`             | '总计'   |
| value  | 中心数值 | `number` \| `string` | 数值总和 |
